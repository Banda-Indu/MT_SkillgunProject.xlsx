Project Description:-

This project focuses on manual testing of the SkillGun web application, where test cases are created and executed to verify different functionalities like login, registration, and user actions.

The test cases are documented in an Excel sheet with:

Test Case ID
Test Case Description
Steps to Execute
🔹 Objective

The main objective is to:

Ensure the application works correctly
Identify bugs or errors
Validate user flows like login and registration
